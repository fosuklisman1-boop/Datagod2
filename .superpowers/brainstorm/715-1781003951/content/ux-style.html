WhatsApp Bot — Navigation Style

How should users navigate the bot menus? This is the biggest UX decision — it affects how every screen is built.

A

Numbered Text Menus

Identical to USSD — user replies 1, 2, 3…

👋 Welcome to Datagod

2. Buy Airtime

4. Results Checker
0. Exit
1

1. MTN

1

Pros

  • Zero extra complexity
  • Re-use all USSD logic as-is
  • Familiar to USSD users

Cons

  • Feels like SMS, not WhatsApp
  • No rich WhatsApp UI
B

Interactive Buttons & Lists

Native WhatsApp UI — tap-to-select menus & reply buttons

Hi
👋 Welcome to Datagod
Choose a service to get started:
≡ View Services
📦 Buy Data Bundle
Select network:
MTN
Telecel
AirtelTigo
AT-iShare

Pros

  • Native WhatsApp feel
  • Tap — no typing numbers
  • Professional & modern

Cons

  • More complex to build
  • Button limit: 3 per message
  • List limit: 10 items per section

💡 Option B uses WhatsApp's Interactive Messages API — buttons for small choices (≤3), list picker for longer menus like bundles. Option A is a direct port of the USSD flow with no USSD-style constraints (no 160-char limit, no 120s timeout).
